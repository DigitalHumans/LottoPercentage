# LottoPercentage
로또 확률
## 규칙
### 커밋 규칙
1. 각자 수정사항이 있으면 브랜치를 만들고 수정한 후에 마스터에 풀 리퀘스트 할 것.
2. 머지 전에 반드시 코드리뷰를 거칠 것.
3. 자신의 풀 리퀘스트를 자신이 머지하지 말 것.
4. 풀 리퀘스트 없이 머지하지 말 것.
5. 담당자가 본인밖에 없는 브랜치로의 머지는 예외.
6. 다른 사람의 승인을 받으면 누구나 머지 가능.
